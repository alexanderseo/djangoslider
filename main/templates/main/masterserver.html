МастерСервер

Зачем нужен МастерСервер?

МастерСервер неотъемлемая часть проекта по раскуртке игровых серверов кс 1.6, ксс. Без него невозможна работа современного мониторинга игровых серверов cs 1.6, css, css v34.

С помощью программы, называемой MasterServer, производится добавление серверов добавленных в мониторинг кс 1.6 в клиент игры. Иными словами, когда игрок нажимает в клиенте кс 1.6 кнопку обновить список серверов, идет запрос к мастерсерверу, который в свою очередь запрашивает у мониторинга игровых серверов какие серверы отобразить в списке игроку.
